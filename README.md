# 📚 Sistema de Empréstimo de Livros em Java

Projeto desenvolvido em **Java** com execução via **console**, com o objetivo de simular o gerenciamento de empréstimos de livros de uma biblioteca.

Este sistema foi criado como prática dos principais fundamentos de programação orientada a objetos, trabalhando conceitos como:

- classes e objetos
- encapsulamento
- composição entre classes
- listas
- interação com o usuário via console
- organização de regras de negócio

---

## 🎯 Objetivo do Projeto

O sistema permite que o usuário:

- visualize os livros disponíveis para empréstimo
- escolha um livro pelo **id**
- informe seu nome
- registre o empréstimo
- torne o livro indisponível após o empréstimo

Ao final, o sistema exibe uma mensagem de encerramento e finaliza a execução.

---

## 🛠️ Tecnologias Utilizadas

- **Java**
- **Programação Orientada a Objetos (POO)**
- **Scanner** para leitura de dados no console
- **ArrayList / List** para armazenamento em memória
- **LocalDate** e **LocalDateTime** para manipulação de datas
